quanthub 是一个量化研究员们的交流群

有人说quant是一群特别孤单的人，这个行业，壁垒层层，很少有人愿意分享，愿意交流。很多时候，你是一个人在研究，一个人去尝试，你永远不知道你为之努力的方向是否能有一个满意的结果，又有哪些坑在等着你。在刚刚踏入这个行业的初期，伴随你的只有孤独和焦虑。运气不好的人，研究方向不合适，一次次重头再来，最终在灰心丧气中离开这个行业。某种程度上来说，做quant不如做码农。
我同意这个观点。但我算是运气比较好的那一个。工作的四年里，跳槽了很多家公司，每一个老板和同事都对我很好，他们愿意把他们经验，观点分享给我。愿意帮我敲定一个比较有希望的方向，帮我指出我研究中的错误。我特别感谢他们，也深深明白，这样的交流与指点对一个研究员的重要性。
现在，轮到我自己去招聘，去培养人了，说实话，好难啊。太多人不了解这个行业，不知道自己要做什么，不知道自己需要哪些技能，或者做错了方向。我就想，为什么不可以，通过一些方式，提高大家之间的交流呢？ 平等，公平的交流，对每个人都是有益的。
在这个群里，你可以交流这个行业未来的发展方向，你可以请求大佬们帮你答疑解惑，你可以，分享自己的一些研究感悟，让大家帮你把把关。当然，你也可以在这里灌灌水，聊一些行业趣事。减轻自己工作压力，让工作更有趣一点。
我们没有太多限制，只希望，你正直善良，对人友善，在寻求索取之前，先去分享
